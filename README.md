# NEXT
Project2 NEXT

A travel app. The app provides users with the ability to plan their travel. Users are required to create an account using passport. Once logged in, users can create a trip and add various destinations. When a trip is saved, users can share the trip details via email. On trip completion, users can add reviews for the trip and based on their experience with food, tours, accommodations etc.

The front page of the app displays 10 most recent reviews of various destinations from our database. Users can also search for reviews for destinations

Technologies used: MVC framework, JavaScript, Node.js, Express, MySQL, Sequelize, Handlebars, passport, various npm packages.
